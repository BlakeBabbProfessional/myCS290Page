Assignment 1

What was challenging?

Figuring out how to size Boxes in a reasonable way was quite challenging. Just setting them to a specific size would cause them to run off the screen if the window size was too small, or to create a huge blank space if there wasn't much content in a Box. Ultimately, I figured out that I could do height: fit-content; to make the Box automatically resize vertically. I could not figure out how to do it horizontally without the page becoming really ugly.

What did I learn?

I already knew the basic structure of HTML and CSS, as I already did a little bit of it back in CS 261 and an into programming class in high school. Despite that, I learned quite a lot over the course of this assignment. I learned how to use the <div> tag, I became more familiar with CSS classes, I learned about the Box model, and on my own I learned about fit-content, <span> and escape characters.

What resources were most helpful?

  • W3 Schools was really helpful for learning about specific concepts and tags in HTML and CSS.
  • StackOverflow was occasionally useful for figuring out some things, but StackOverflow's comments are often so elitist and mean that it made me reluctant to visit it.
  • Surprisingly, the most helpful resource (besides the lectures obviously) was my IDE, IntelliJ Ultimate, which I got through GitHub Education. It really helped me to format my code and find errors and was constantly giving me suggestions on improvements and how to accomplish tasks. It even has text explaining what every single tag does!
